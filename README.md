## 一个青岛OJ[OnlineJudge](https://github.com/QingdaoU/OnlineJudge)的命令行版！
## 安装
1. 下载此代码
2. 首次使用请自行安装 Python 和 pip 并执行以下命令
```
pip install --upgrade pip
pip install -r requirements.txt
```
3. 运行
```
python 稳健OJ命令行版.py
```
